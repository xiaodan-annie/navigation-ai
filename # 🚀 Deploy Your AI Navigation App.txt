---

# 🥇 PART 1 — Prepare Backend (FastAPI)

---

# 📁 Step 1 — Open backend folder

Example:

```text id="folder"
navigation-ai/backend
```

---

# 📦 Step 2 — Create requirements.txt

Inside backend terminal:

```bash id="requirements"
pip freeze > requirements.txt
```

This creates:

```text id="req_file"
requirements.txt
```

---

# 📦 Step 3 — Create render.yaml (optional but professional)

Inside backend folder create:

```text id="render_yaml"
render.yaml
```

Add:

```yaml id="yaml_content"
services:
  - type: web
    name: navigation-ai-backend
    env: python
    buildCommand: pip install -r requirements.txt
    startCommand: uvicorn main:app --host 0.0.0.0 --port $PORT
```

---

# 🥈 PART 2 — Push project to GitHub

Render and Netlify deploy easiest from GitHub.

---

# ✅ Step 4 — Install Git

Install:

[Git for Windows](https://git-scm.com/download/win?utm_source=chatgpt.com)

---

# ✅ Step 5 — Create GitHub account

[GitHub](https://github.com?utm_source=chatgpt.com)

---

# ✅ Step 6 — Initialize Git repo

Open terminal at project root:

```bash id="git_init"
git init
```

---

# ✅ Step 7 — Create .gitignore

Create file:

```text id="gitignore"
.gitignore
```

Add:

```text id="gitignore_content"
__pycache__/
.env
venv/
```

---

# ✅ Step 8 — Commit project

```bash id="git_commit"
git add .
git commit -m "Initial deployment"
```

---

# ✅ Step 9 — Push to GitHub

Create empty GitHub repo:
Example:

```text id="repo_name"
navigation-ai
```

Then run:

```bash id="git_push"
git remote add origin YOUR_GITHUB_REPO_URL
git branch -M main
git push -u origin main
```

---

# 🥉 PART 3 — Deploy Backend to Render

---

# ✅ Step 10 — Create Render account

[Render](https://render.com?utm_source=chatgpt.com)

Sign in with GitHub.

---

# ✅ Step 11 — Create Web Service

Dashboard:

```text id="render_steps"
New +
→ Web Service
→ Connect GitHub repo
```

Select your repo.

---

# ✅ Step 12 — Configure backend

Use:

| Setting        | Value                                          |
| -------------- | ---------------------------------------------- |
| Root Directory | backend                                        |
| Runtime        | Python                                         |
| Build Command  | `pip install -r requirements.txt`              |
| Start Command  | `uvicorn main:app --host 0.0.0.0 --port $PORT` |

---

# ✅ Step 13 — Add environment variables

In Render:

```text id="env_section"
Environment → Add Environment Variable
```

Add:

| Key            | Value    |
| -------------- | -------- |
| OPENAI_API_KEY | your key |
| ORS_API_KEY    | your key |

---

# ✅ Step 14 — Deploy backend

Click:

```text id="deploy"
Deploy Web Service
```

After deploy you get URL like:

```text id="backend_url"
https://navigation-ai-backend.onrender.com
```

SAVE THIS.

---

# 🏅 PART 4 — Deploy Frontend to Netlify

---

# ✅ Step 15 — Create Netlify account

[Netlify](https://www.netlify.com?utm_source=chatgpt.com)

---

# ✅ Step 16 — Update frontend API URL

Inside `app.js`

REPLACE:

```javascript id="localhost_replace"
http://127.0.0.1:8000/route
```

WITH:

```javascript id="production_url"
https://YOUR_RENDER_URL.onrender.com/route
```

Example:

```javascript id="real_example"
https://navigation-ai-backend.onrender.com/route
```

---

# ✅ Step 17 — Deploy frontend

In Netlify:

```text id="netlify_steps"
Add new site
→ Import from GitHub
→ Select repo
```

---

# ✅ Step 18 — Configure frontend

If frontend folder separate:

| Setting           | Value         |
| ----------------- | ------------- |
| Base directory    | frontend      |
| Build command     | (leave empty) |
| Publish directory | frontend      |

---

# ✅ Step 19 — Deploy

Click:

```text id="deploy_site"
Deploy Site
```

Netlify gives URL like:

```text id="frontend_url"
https://navigation-ai.netlify.app
```
